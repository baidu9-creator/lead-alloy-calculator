# Lead Alloy Calculator

Cast bullet and lead alloy composition tool. Built with React + Vite, deployed via GitHub Pages.

**Live app:** https://baidu9-creator.github.io/Lead-alloy-calculator/

## Features
- 32 preset alloys with known Brinell hardness
- Mix mode — blend alloys by weight with manual price entry
- Real-time composition breakdown and donut chart
- Boolit count and cost calculator
- iOS home screen installable (PWA-ready)

## Deploy
Push to `main` — GitHub Actions builds and deploys automatically.

## Local dev
```bash
npm install
npm run dev
```
